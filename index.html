Why YorunoBrowser?

Safariが悪いんじゃない。
夜の使い方に、合ってないだけ。

大きいiPhone、白すぎるサイト、上にあるボタン、広告で重いページ。YorunoBrowserは、その小さなストレスを夜用に作り直したブラウザです。

MAIN FEATURE

親指の届く場所に、全部置いた。

戻る、進む、タブ、更新、検索。よく使う操作は画面下に集約。上端に指を伸ばしてスマホを落とす日々を終わらせます。

白いサイトを、夜にする。

強制ダークモードとDarkerトグルで、深夜の眩しさを抑えます。最低輝度でも眩しい人向け。

漫画サイトを、軽く読む。

重いJavaScriptや広告に邪魔されず、画像ビューアのように読み進められるComicReaderを搭載。

保存したものを、迷子にしない。

ブラウザ内にファイル管理を統合。動画、画像、音声をローカルで整理しやすくします。

ComicReader

広告だらけの漫画サイトを、
読むことに集中できる画面へ。

YorunoBrowserは「普通のブラウザ」ではなく、夜にメディアを見る人のための道具です。ComicReaderは、その中心機能のひとつ。

  • 画像中心のページを読みやすく表示
  • 余計な広告・オーバーレイを減らす
  • 発熱と重さを抑えた閲覧体験
  • 片手操作のままページ移動

Save Locally

見つけたものを、
ローカルで整理する。

対応サイトでは、動画・音声・画像を端末内に保存できます。DRMやログイン制限を回避するための機能ではありません。利用するサイトの規約と法律の範囲で使ってください。

  • MP4 / HLS / 画像などに対応
  • 失敗したダウンロードは原則カウント消費なし
  • YouTube / Netflix / Amazon Prime VideoなどのDRM系は非対応
片手操作も、強制ダークも、無料。

コア機能はFreeプランで使えます。課金が効いてくるのは「もっと保存したい」「応援したい」と思ってから。詳細はアプリ内Billing画面で。

AltStore PALで入れる

  1. AltStore PALをインストールまず altstore.io からAltStore PAL本体をiPhoneに入れる。
  2. YorunoBrowserのソースURLをコピー下のボタンでコピーできます。
    altstore://source?url=https://raw.githubusercontent.com/C0A2518617/YorunoBrowser_Official/main/source.pal.json
  3. Safariのアドレスバーに貼って開くペーストして実行 →「開く」をタップすると、AltStoreにソースが追加されます。
  4. AltStoreからインストール一覧のYorunoBrowserをインストールしたら完了。以後はAltStoreからアップデートできます。
🌍 EU/日本以外の地域の人はこちらVIA ALTSERVER — 手動アップデート
  1. PCに AltServer をインストール
  2. iPhoneにAltStore Classicをインストール
  3. GitHub Releases から .ipa をダウンロード
  4. AltStore経由でIPAをインストール

この方式はアップデートが手動です。新しい.ipaを毎回インストールしてください。

FAQ

入れる前に、これだけ。

無料で使える?

使えます。片手操作、強制ダークモード、ComicReaderなどのコア機能はFreeプランで使えます。

なんでApp Storeにないの?

一部機能がApp Storeの審査と相性が悪いため、AltStore経由で配布しています。

データを抜いてる?

閲覧履歴や保存ファイルを売るためのアプリではありません。保存したファイルは端末内で管理されます。詳しくはプライバシーポリシーを確認してください。

違法じゃないの?

YorunoBrowserはブラウザとファイル管理ツールです。利用するサイトの規約と法律の範囲で、自分が扱う権利のあるコンテンツに使ってください。

Android / Windows / Mac版は?

現在はiPhoneのみです。将来的には検討するかもしれませんが、今はiPhone版に集中しています。
