Why 4Me?

Safariが悪いんじゃない。
片手で読むには、少し遠いだけ。

大きいiPhone、上にあるボタン、広告で重いページ。4Meは、その小さなストレスを片手操作と読みやすさで整えるブラウザです。

MAIN FEATURE

親指の届く場所に、全部置いた。

戻る、進む、タブ、更新、検索。よく使う操作は画面下に集約。上端に指を伸ばしてスマホを落とす日々を終わらせます。

読みやすい明るさに整える。

サイトの眩しさやコントラストを抑えたいとき、表示をすばやく調整できます。

漫画サイトを、軽く読む。

重いJavaScriptや広告に邪魔されず、画像ビューアのように読み進められるComicReaderを搭載。

ComicReader

広告だらけの漫画サイトを、
読むことに集中できる画面へ。

4Meは、画像中心のページを読みやすくするためのブラウザです。ComicReaderは、その中心機能のひとつ。

  • 画像中心のページを読みやすく表示
  • 余計な広告・オーバーレイを減らす
  • 発熱と重さを抑えた閲覧体験
  • 片手操作のままページ移動

RECOMMENDED — 🇯🇵 日本 / 🇪🇺 EU のユーザー

AltStore PALで入れる

  1. AltStore PALをインストールまず altstore.io からAltStore PAL本体をiPhoneに入れる。
  2. 4MeのソースURLをコピー下のボタンでコピーできます。
    altstore://source?url=altstore://source?url=https://raw.githubusercontent.com/C0A2518617/4Me_Official/main/source.pal.json
  3. Safariのアドレスバーに貼って開くペーストして実行 →「開く」をタップすると、AltStoreにソースが追加されます。
  4. AltStoreからインストール一覧の4Meをインストールしたら完了。以後はAltStoreからアップデートできます。
🌍 EU/日本以外の地域の人はこちらVIA ALTSERVER — 手動アップデート
  1. PCに AltServer をインストール
  2. iPhoneにAltStore Classicをインストール
  3. GitHub Releases から .ipa をダウンロード
  4. AltStore経由でIPAをインストール

この方式はアップデートが手動です。新しい.ipaを毎回インストールしてください。

FAQ

入れる前に、これだけ。

無料で使える?

使えます。片手操作、表示調整、ComicReaderなどのコア機能はFreeプランで使えます。

配布方法は?

AltStore経由で配布しています。上の手順どおりにソースを追加するとインストールできます。

Android / Windows / Mac版は?

現在はiPhoneのみです。将来的には検討するかもしれませんが、今はiPhone版に集中しています。
